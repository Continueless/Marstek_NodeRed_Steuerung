# Marstek NodeRed Steuerung

Steuert Marstek Speichr insbesondere wenn man mehr als nur einen Spericher hat. Dies ist die einzige möglichkeit praktisch Verlustfrei mehere Marstek Speicher zu steuern. Theoretisch kann man so unendlich viele Marstek Speicher an nur einem zähler mit nur einem einzigen Meßgerät steuern.

Getestest mit Marstek Venus E 3.0

theoretisch funktioniert dieses Script mit jedem Marstek Speicher. Gerne informarion, dazu ob es mit anderen Typen klappt. Register können von Speicher zu Speicher leicht abweichen.

Die entsprechenden Variablen für z.b. Aktueller Stromverbrauch müssen angepasst werden, je nachdem wie dieser bei Dir ausgelsen wird (Shelly, Hichi, ...)
Credentials der Modbusschnittstelle des Speichers müssen angegeben werden.
